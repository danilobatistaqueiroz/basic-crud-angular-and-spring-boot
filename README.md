# Basic CRUD with Angular 5 and Spring Boot

This project uses:  

In Angular  
* NgForm (@angular/forms)
* rxjs/Subscription
* rxjs/Observable
* ActivatedRoute (@angular/router)
* HttpClient (@angular/common/http)

In Java  
* Lombok
* JPA
* JpaRepository
* CrossOrigin (CORS)
